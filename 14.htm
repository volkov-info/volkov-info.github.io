МОЯ РАБОТА

Из года в год одиннадцатиклассники выбирают для сдачи информатику. Подготовка к сдаче экзамена начинается на обычных уроках, когда при изучении той или иной темы, большое внимание уделяется вопросам как эта тема проверяется на экзамене, какие типы задач по данной теме существуют, и какие есть алгоритмы их решения. Задачи, предлагаемые к решению в классе и дома максимально приближены к экзаменационным задачам прошлых лет экзамена.

Для углубленного изучения языка программирования для учащихся 9-10 классов ведется спецкурс. Одну из презентаций для спецкурса можно посмотреть здесь.

Для одиннадцатикласников также ведется спецкурс по подготовке к экзамену. Материалы одного из занятий этого года можно увидеть здесь.

Результаты сдачи нашими лицеистами экзамена, обычно на одном уровне, а чаще превышают показатели по району.
